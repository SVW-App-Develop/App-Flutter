# 9단계 : 전자액자
- 위젯 생명주기, PageView, Timer, SystemChrome, StatefulWidget

  - PageView 위젯 : 가로 또는 세로로 스와이프해 화면에 보이는 위젯을 변경
 
  - Timer : 일정 기간을 기다리거나 주기적으로 반복 작업을 실행하는 데 사용
