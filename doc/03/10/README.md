# 10단계 : D-Day
- 상태 관리, CupertinoDatePicker, Dialog, DateTime

  - StatefulWidget 이용한 상태 관리
 
    - setState() 함수를 사용한 상태 관리
   
  - 쿠퍼티노(cupertino) 위젯
 
    - 다이얼로그(dialog) 및 데이터픽커(datepicker) 를 Cupertino 위젯을 사용해 구현
 
- 플러터의 두 가지 디자인 시스템

  - Marterial 위젯 : 구글의 머티리얼 디자인 기반
 
  - Cuperino 위젯 : iOS 스타일의 디자인
